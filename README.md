# parenting-seminar-lp

子育てセミナー用ランディングページです。

## 公開URL

https://twinkbaby-lang.github.io/parenting-seminar-lp/

## 画像の置き場所

画像は `images/` フォルダに配置します。
HTML内の画像パスは相対パスで指定しています。

- images/hero_mother_child.webp
- images/worried_mother.webp
- images/lecturer.webp
- images/gift_materials.webp
- images/consultation.webp
- images/voice_mama_6.webp
- images/voice_mama_4.webp
- images/voice_mama_8.webp

## 差し替え箇所

`index.html` 内の `FORM_URL_HERE` を申込フォームURLに差し替えてください。
